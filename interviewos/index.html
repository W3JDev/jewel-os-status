Features

What's Built

Confirmed capabilities as of April 2026.

🤖

AI Candidate Scoring

LIVE

Claude-powered scoring engine evaluates candidates against job requirements. Opus handles complex decisions; Haiku runs background cron evaluations.

📊

CV / Resume Ranker

IN REVIEW

Weighted scoring: 50% skill fit, 20% nice-to-haves, 20% industry experience, 10% communication. Pending Lenore & Abhinanda approval.

🔗

LinkedIn Job Posting

LIVE

Direct LinkedIn job posting integration — solved and live. One-click publish from within the ATS. Volume limit testing underway.

📤

Application Intake Pipeline

LIVE

End-to-end candidate intake — form submission, CV ingestion, initial AI triage, and queue management into recruiter workflow.

🔍

AI-Driven Sourcing

LIVE

Brave API discovers candidates, Clay enriches with email + location, Apollo acts as fallback. Fully autonomous talent pipeline.

💌

Daily Digest Emails

LIVE

Automated daily digest delivering pipeline summaries and sourcing activity to the BAIIT team. Running and confirmed delivering.

📈

Agent Cost/Value Dashboard

IN PROGRESS

Tracks AI spend vs value delivered. Anthropic API is source of truth. Co-owned with Ron Rivero and Melanie Goff.

🤖

Zara Agent Integration

IN PROGRESS

Zara (Bold Business AI agent) integration into ATS workflow for autonomous recruiter actions. Architecture defined; implementation underway.

Architecture

System Stack

React/Next.js frontend, FastAPI backend, Supabase for data — migrating from Lovable to AWS EC2 this week.

Frontend
React / Next.js
Originally built on Lovable. Migrating to self-hosted AWS EC2. TypeScript + Tailwind.
Database
Supabase
PostgreSQL backend. Exporting from Lovable — DB, edge functions, secrets + RSL policies. Migration in progress.
Backend
FastAPI (Python)
REST API layer for AI agent orchestration, CV processing, sourcing pipeline, and digest generation.
AI / Agents
Claude API
Opus for scoring decisions, Sonnet for code/analysis, Haiku for cron jobs. Anthropic API = cost source of truth.
Hosting
AWS EC2
Migrating from Lovable this week. Staging + prod environments being configured.
Source Control
GitHub
W3JDev/interviewos. CI/CD via GitHub Actions.
